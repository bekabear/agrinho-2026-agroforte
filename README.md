###🌿 AgroForte - Tecnologia e Sustentabilidade no Campo

## 📖 Sobre o Projeto

O **AgroForte** é um site desenvolvido para o **Programa Agrinho 2026**, com o objetivo de demonstrar como a tecnologia pode transformar a agricultura e diversos setores da sociedade, promovendo inovação, eficiência e sustentabilidade.

O projeto apresenta uma navegação moderna e intuitiva, mostrando exemplos de tecnologias inteligentes aplicadas ao campo, além de soluções sustentáveis que contribuem para a preservação do meio ambiente e para uma produção agrícola mais eficiente.

---

# 🎯 Objetivo

Demonstrar, de forma visual e interativa, como a tecnologia pode melhorar a qualidade de vida das pessoas e tornar a agricultura mais sustentável, abordando temas como:

* Agricultura Inteligente
* Saúde Digital
* Transporte Inteligente
* Educação Digital
* Indústria 4.0
* Comunicação Digital

Além disso, o projeto apresenta soluções voltadas para a sustentabilidade no campo, incentivando o uso consciente dos recursos naturais.

---

# 🌱 Soluções Sustentáveis Apresentadas

* ☀️ Energia Solar
* 💧 Reaproveitamento da Água
* 🌿 Agricultura Sustentável
* ⚡ Máquinas Elétricas
* 🛰️ Agricultura de Precisão
* ♻️ Redução de Desperdício

# 🚀 Funcionalidades

O site possui diversos recursos desenvolvidos com HTML, CSS e JavaScript, proporcionando uma experiência dinâmica ao usuário.

## Interface

* Layout moderno e responsivo
* Navegação entre páginas
* Hero Banner com imagem de destaque
* Cards interativos
* Estatísticas ilustrativas
* Rodapé personalizado

## Recursos JavaScript

* Alternância entre modo claro e modo escuro
* Armazenamento da preferência do usuário utilizando LocalStorage
* Carrossel automático de imagens
* Manipulação do DOM para alteração de elementos da página
* Menu responsivo para dispositivos móveis

## Recursos CSS

* Flexbox e Grid Layout
* Media Queries para adaptação em celulares e tablets
* Transições suaves
* Efeitos Hover
* Variáveis CSS para gerenciamento do tema
* Sombras e efeito Glassmorphism
